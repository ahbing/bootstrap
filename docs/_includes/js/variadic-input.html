Usage

Via data attributes

The data-ride="variadic-input" attribute is used to mark a element is a variadict input and initialize it at page load. It cannot be used in combination with (redundant and unnecessary) explicit JavaScript initialization of the same variadic input.

{% highlight html %}
{% endhighlight %}

Via JavaScript

Call a variadic input with id myVariadicInput with a single line of JavaScript:

{% highlight js %}$('#myVariadicInput').variadicInput(options){% endhighlight %}

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-placehoder="".

Name type default description
placehoder string 'extra parameter' Use as placehoder of dynamic added input.
