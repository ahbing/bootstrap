Usage

Via JavaScript

Call a loading with id myBtn with a single line of JavaScript:

{% highlight js %}$('#myBtn').loading('start').{% endhighlight %}

Options

Options can be passed via JavaScript.

Name type default description
duration number 1000 Minimum milliseconds loding animation duration.

Methods

.loading('start')

Manually start a loaing.

.loading('stop')

Manually stop a loaing.

.loading(options)

Activates your content as a loading. Accepts an optional options object.

{% highlight js %} $('#mySumitBtn').loading({ duration: 800 }) {% endhighlight %}
